_banner.twig

One wide banner view

If you select a banner block as the source for this banner the htmllink field will be used as a link.

If you select any other content type, the banner will link to the full page with that content.

_bannertriple.twig

Three banners view

If you select banner blocks as the source for this banner the htmllink field will be used as a link.

If you select any other content type, the banner will link to the full page with that content.

_title.twig

Title with a link

This template can be used for very simple lists of pages. It is a plain HTML list with <UL> and <LI> elements.

_teaser.twig

Teaser view

Use this template for the normal view of teasers in a listing.

_listing.twig

Listing of records

Make a listing page with the selected content type.

_listingnews.twig

Two news postings and a link to more

Make a block with two columns of records from a chosen content type.

_listingtable.twig

Person listing as a table

Show a table of records from the persons content type.

_listingsemistatic.twig

Three static intro item

Make a block with three columns of records from a chosen content type.

_listingthumbnails.twig

Four columns with round thumbnailed links

Make a block with four columns of records from a chosen content type.

_detail.twig

Full details for single items

Show the the full page from a record in one module.

_breaker.twig

One section breaker, call to action

If you select a banner block as the source for this breaker the htmllink field will be used as a link.

If you select any other content type, the breaker will link to the full page with that content.

_breakerdouble.twig

Two section breakers, call to action

If you select banner blocks as the source for this breaker the htmllink field will be used as a link.

If you select any other content type, the breaker will link to the full page with that content.

_streamercolumn.twig

One column category streamer

You can use one, two or three category streamer columns to place the columns next to eachother.

_calltoactionlinks.twig

Call to action button main page

If you select banner blocks as the source for this call to action button the htmllink field will be used as a link.

If you select any other content type, the call to action button will link to the full page with that content.

_calltoactionblog.twig

Call to action blog page

If you select banner blocks as the source for this call to action button the htmllink field will be used as a link.

If you select any other content type, the call to action button will link to the full page with that content.

_filecontent.twig

Single file attachment block

Choose one file to add a download module for that file.

_fileslisting.twig

File list block

Choose one directory to add a download module for all files in that directory.

_body.twig

Module body field

For extra body blocks between modules.

_collapsedcontent.twig

Collapsed content

A block of content that opens with a click on the header.

_image.twig

One wide image view

Show a simple image over the full width of the content.
